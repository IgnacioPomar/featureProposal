# Contexto para Codex

Esta carpeta guarda el contexto operativo del proyecto para evitar repetirlo en cada prompt.

Uso recomendado al iniciar una sesión:

1. "Revisa `.ai/contexto-proyecto.md` y `.ai/checklist-poc.md`."
2. "Trabaja siguiendo ese contexto."
3. Si cambian objetivos, actualiza primero esos archivos.

Nota:
- No existe memoria persistente automática entre sesiones.
- La forma estándar y fiable es guardar el contexto en archivos del repo (como esta carpeta).
